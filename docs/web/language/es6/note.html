    # Set,Map

    # 1. Set

    set和数组类似，但是里面元素是唯一的

    set不会改变数据的类型，数字5，字符串5是不相同的
    set判断是否相同类似 ===，但是NaN它认为是一样的
    对象总是不相等的

[...new Set(Array)]
// 去除字符串的重复元素
[...new Set('aassew')].join('')

//例子2
let set = new Set()
set.add({})

    # 2. WeakSet

    • WeakSet 的成员只能是对象，而不能是其他类型的值
    • WeakSet 中的对象都是弱引用，即垃圾回收机制不考虑 WeakSet 对该对象的引用
    • WeakSet 适合临时存放一组对象，以及存放跟对象绑定的信息。只要这些对象在外部消失，它在 WeakSet 里面的引用就会自动消失。
    • WeakSet 不可遍历

    # 语法

    WeakSet 可以接受一个数组或类似数组的对象作为参数

    new WeakSet([[1,2]])
new WeakSet([1,2]) // error 要数组里面的值是对象
    1
    2

    方法

    • WeakSet.prototype.add(value)：向 WeakSet 实例添加一个新成员。
    • WeakSet.prototype.delete(value)：清除 WeakSet 实例的指定成员。
    • WeakSet.prototype.has(value)：返回一个布尔值，表示某个值是否在 WeakSet 实例之中。

    没有size，不能遍历

    实用例子
    确定一个方法只能实例调用

    const foos = new WeakSet()
class Foo {
  constructor() {
    foos.add(this)
  }
  method () {
    if (!foos.has(this)) {
      throw new TypeError('Foo.prototype.method 只能在Foo的实例上调用！');
    }
  }
}

    # 结合数组的map方法、filter方法，可以实现 Map 的遍历和过滤

    const map0 = new Map()
  .set(1, 'a')
  .set(2, 'b')

const map1 = new Map(
  [...map0].filter(([k, v]) => k < 3)
);

const map2 = new Map(

    # 4. WeakMap

    WeakMap只有四个方法可用：get()、set()、has()、delete()。
    不可以遍历，弱引用

    # WeakMap用途

    主要作用就是不会造成内存泄漏

    1. 典型场合就是 DOM 节点作为键名
    let myWeakmap = new WeakMap();

myWeakmap.set(
  document.getElementById('logo'),
  {timesClicked: 0})
;

document.getElementById('logo').addEventListener('click', function() {
  let logoData = myWeakmap.get(document.getElementById('logo'));
  logoData.timesClicked++;
}, false);

    # 6. FinalizationRegistry

    用来指定目标对象被垃圾回收机制清除以后，所要执行的回调函数

    const registry = new FinalizationRegistry(heldValue => {
  // ....
});

    注册表实例的register()方法，用来注册所要观察的目标对象。

    registry.register(theObject, "some value")
// theObject就是所要观察的目标对象，一旦该对象被垃圾回收机制清除，注册表就会在清除完成后，调用早前注册的回调函数，并将some value作为参数（前面的heldValue）传入回调函数

    例子

    function makeWeakCached(f) {
  const cache = new Map();
  const cleanup = new FinalizationRegistry(key => {
    const ref = cache.get(key);
    if (ref && !ref.deref()) cache.delete(key);
  });

  return key => {
    const ref = cache.get(key);
    if (ref) {
      const cached = ref.deref();
      if (cached !== undefined) return cached;
    }

    const fresh = f(key);
    cache.set(key, new WeakRef(fresh));
    cleanup.register(fresh, key);
    return fresh;
  };
}

const getImageCached = makeWeakCached(getImage);
