    # 53. 最大子序和

    输入：nums = [-2,1,-3,4,-1,2,1,-5,4] 输出：6 解释：连续子数组 [4,-1,2,1] 的和最大，为 6 。

    思路:

    1. 使得每个元素之前的和要是大于0的，要是大于0就继续加元素本身，如果小于0就从元素本身开始计算。
    2. 最大值就从元素和中比较取最大值 第一个条件就相当于将数组分割成连续的一段段的数组，数组分割依据是总和小于零，所以被分割的数组中的元素前的总和肯定是大于0的

    # 88. 合并两个有序数组

    非递减顺序 排列的整数数组 nums1 和 nums2，另有两个整数 m 和 n ，分别表示 nums1 和 nums2 中的元素数目. 请你 合并 nums2 到 nums1 中，使合并后的数组同样按 非递减顺序 排列。 注意：最终，合并后数组不应由函数返回，而是存储在数组 nums1 中。为了应对这种情况，nums1 的初始长度为 m + n，其中前 m 个元素表示应合并的元素，后 n 个元素为 0 ，应忽略。nums2 的长度为 n 。

    思路：

    1. num1的长度是合并的长度，将num2从num1后面比较，num1尾部数字（除0）和num2尾部比较
    2. num2大就直接替换，num1大就交换
    3. 滑动索引
    4. 以num2的索引为-1结束
    var merge = function(nums1, m, nums2, n) {
    let indexA = m - 1
    let indexB = n - 1
    if(!nums2.length){
        return nums1
    }
    while(indexB!=-1){
        let a = nums1[indexA]
        let b = nums2[indexB]
        if(a < b || indexA==-1){
            nums1[indexA + indexB + 1] = b
            indexB--
        }else{
            nums1[indexA + indexB + 1] = a
            nums1[indexA] = 0
            indexA--
        }
    }
    return nums1
};

    var isSymmetric = function(root) {
    if(root.left?.val !== root.right?.val){
        return false
    }
    return aa(root.left, root.right)

var aa = function(leftNode, rightNode) {
    let leftResult = true
    let rightResult = true
    if(leftNode?.val !== rightNode?.val){
        return false
    }
    if(leftNode?.left || rightNode?.right){
        leftResult = aa(leftNode.left,rightNode.right)
    }
    if(leftNode?.right || rightNode?.left){
        rightResult = aa(leftNode.right,rightNode.left)
    }
    return leftResult && rightResult
}

    # 108. 将有序数组转换为二叉搜索树

    给你一个整数数组 nums ，其中元素已经按 升序 排列，请你将其转换为一棵 高度平衡 二叉搜索树。

    高度平衡 二叉树是一棵满足「每个节点的左右两个子树的高度差的绝对值不超过 1 」的二叉树。

    思路：

    1. 中序遍历二叉搜索树数组是升序的，现在就是要将升序数组用中序的方式还原二叉树
    2. 又是高度平衡的，就从数组中间取值，当作根结点
    var sortedArrayToBST = function(nums) {
    let treeNode = {}
    let minIndex = Math.floor(nums.length/2)
    treeNode.val = nums[minIndex]
    treeNode.left =  minIndex>0 ? sortedArrayToBST(nums.slice(0,minIndex)) : null
    treeNode.right = minIndex<nums.length-1 ? sortedArrayToBST(nums.slice(minIndex+1,nums.length)) : null
    return treeNode
};

    var isBalanced = function(root) {

var isRecur = function(root){
    if(!root){
        return 0
    }
    let leftDeep = isRecur(root.left)
    if(leftDeep === -1){
        return -1
    }
    let rightDeep = isRecur(root.right)

    }
    return Math.abs(leftDeep - rightDeep) < 2 ? Math.max(leftDeep, rightDeep) + 1 : -1

    # 111. 二叉树的最小深度 !!!

    给定一个二叉树，找出其最小深度。

    最小深度是从根节点到最近叶子节点的最短路径上的节点数量。

    说明：叶子节点是指没有子节点的节点。

    var minDepth = function(root) {
    if(!root){
        return 0
    }
    let left = minDepth(root.left)
    let right = minDepth(root.right)
    return !root.left || !root.right ? left + right +1 : Math.min(left,right)+1
};

    # 112. 路径总和 !!!

    二叉树的根节点 root 和一个表示目标和的整数 targetSum ，判断该树中是否存在 根节点到叶子节点 的路径，这条路径上所有节点值相加等于目标和 targetSum 。

    来源：力扣（LeetCode） 链接：https://leetcode-cn.com/problems/path-sum 著作权归领扣网络所有。商业转载请联系官方授权，非商业转载请注明出处。

    var hasPathSum = function(root, targetSum) {
    if(!root){
        return false
    }
    if(!root.left && !root.right){
        return root.val === targetSum
    }
    return hasPathSum(root.left, targetSum - root.val) || hasPathSum(root.right, targetSum - root.val)
};

    # 121. 买卖股票的最佳时机

    给定一个数组 prices ，它的第 i 个元素 prices[i] 表示一支给定股票第 i 天的价格。

    你只能选择 某一天 买入这只股票，并选择在 未来的某一个不同的日子 卖出该股票。设计一个算法来计算你所能获取的最大利润。

    返回你可以从这笔交易中获取的最大利润。如果你不能获取任何利润，返回 0 。

    输入：[7,1,5,3,6,4]
输出：5
输入：prices = [7,6,4,3,1]
输出：0
    1
    2
    3
    4

    思路：
    实际上就是找最大的差值，转换一下就是找一段区间的最大最小值，所以遍历数组的时候记录下 最小的数字，然后用最小的数字来减遍历的数字，得到差值，存下最大的差值就ok

    var maxProfit = function(prices) {
    let minPrice = 10**4 + 1
    let maxDiff = 0
    for(let price of prices){
        minPrice = Math.min(minPrice, price)
        maxDiff = Math.max(maxDiff, price - minPrice)
    }
    return maxDiff
};

    # 136. 只出现一次的数字

    给定一个非空整数数组，除了某个元素只出现一次以外，其余每个元素均出现两次。找出那个只出现了一次的元素。 思路
    只有一个落单的数字，而俩数异或，相同的数异或是0，0与一个数异或，是那个数

    var singleNumber = function(nums) {
    let single = 0
    for(let num of nums){
        single ^=num
    }
    return single
};

    # 160. 相交链表

    给你两个单链表的头节点 headA 和 headB ，请你找出并返回两个单链表相交的起始节点。如果两个链表没有交点，返回 null 思路：
    pA走过的路径为A链+B链

    pB走过的路径为B链+A链

    pA和pB走过的长度都相同，都是A链和B链的长度之和，相当于将两条链从尾端对齐，如果相交，则会提前在相交点相遇，如果没有相交点，则会在最后相遇。

    pA:1->2->3->4->5->6->null->9->5->6->null
pB:9->5->6->null->1->2->3->4->5->6->null
    1
    2
    var getIntersectionNode = function(headA, headB) {
    if(!headA || !headB){
        return null
    }
    let nodeA = headA
    let nodeB = headB
    while(nodeA != nodeB){
        nodeA = nodeA ? nodeA.next : headB
        nodeB = nodeB ? nodeB.next : headA
    }
    return nodeA
}
