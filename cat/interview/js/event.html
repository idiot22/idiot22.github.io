    说说你对事件循环的理解

    二喵 1/15/2022

    # 宏任务与微任务

    异步任务划分为宏任务和微任务

    # 微任务

    微任务执行在宏任务之前

    • Promise.then
    • MutaionObserver
    • Object.observe（已废弃；Proxy 对象替代）
    • process.nextTick（Node.js）
    • await下面的语句

    # 宏任务

    • script (可以理解为外层同步代码)
    • setTimeout/setInterval
    • UI rendering/UI事件
    • postMessage、MessageChannel
    • setImmediate、I/O（Node.js）

    # async, await

    await下面的语句都会被放到微任务队列中

    async function fn1 (){
    console.log(1)
    await fn2() // 正常执行
    console.log(2) // 阻塞
}

async function fn2 (){
    console.log('fn2')
}

fn1()
console.log(3)

    # 分析

    async function async1() {
    console.log('async1 start')
    await async2()
    console.log('async1 end')
}
async function async2() {
    console.log('async2')
}
console.log('script start')
setTimeout(function () {
    console.log('settimeout')
})
async1()
new Promise(function (resolve) {
    console.log('promise1')
    resolve()
}).then(function () {
    console.log('promise2')
})
console.log('script end')
